Visualizador ICA – Sistema de visualización territorial

Pasos para usar:
1. Haga doble clic en el archivo "VisualizadorICA.exe"
2. El sistema abrirá automáticamente su navegador predeterminado
3. Cargue su archivo Excel (.xlsx) y disfrute del mapa interactivo

Importante:
• No se requiere instalar Python ni software adicional
• No se debe ejecutar ningún archivo ".bat"
• Este software está empaquetado y listo para su uso en ambientes bloqueados
